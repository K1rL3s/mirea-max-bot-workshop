### Терминология

Немного терминов, чтобы я дальше не писал расшифровку слов

Юзернейм (тэг, ник) - уникальное имя юзера/канала/чата, которое начинается с `@`
ТГ - Телеграм, Telegram
Макс - MAX, Мах
Диалог - приватный чат, личка, директ
Организация - ИП или юрлицо
